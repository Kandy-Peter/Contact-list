About Contaxts

Contaxts is a application that helps users to keep track of contact information of people in their circle eg phone numbers

Required Features

  1. Users can create an acount and login
  2. Users can create a contact
  3. Users can update a contact
  4. Users can favorite a contact
  5. Users can see the details of contact
  6. Users can delete contact
  7. Users should see the list of their contacts

The first challenge requires to create UI templates using HTML, CSS and Javascript.

NB: Any frameworks(Bootstrap, SASS/SCSS) or an already built websites templantes can be used
